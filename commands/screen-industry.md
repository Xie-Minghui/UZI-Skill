# 从行业筛选股票#

筛选指定行业中推荐买入的股票。#

## 用法##

```
/stock-deep-analyzer:screen-industry <行业名称> [--criteria <标准>]
```

## 参数##

- `<行业名称>`: 行业名称（如"白酒"、"新能源"）
- `--criteria`: 筛选标准（可选）
  - `value`: 价值型（低 PE + 高股息 + 护城河）
  - `growth`: 成长型（高增速 + 高 PEG）
  - `balanced`: 均衡型（综合评分 TOP 3）**默认**
  - `aggressive`: 激进型（游资偏好 + 技术突破）

## 示例##

```
/stock-deep-analyzer:screen-industry 白酒
/stock-deep-analyzer:screen-industry 新能源 --criteria growth
/stock-deep-analyzer:screen-industry 医药 --criteria value
```

## 输出##

- **TOP 3 推荐股票**（股票名称、代码、评分、推荐理由）
- **建议配置比例**（每只股票的推荐仓位）
- **行业展望**（看多/看空/中性）
- **风险提示**

## 筛选标准解释##

### `value` - 价值型###

**适合**：稳健投资者、长期投资

**筛选逻辑**：
1. PE < 30
2. PB < 3
3. 股息率 > 2%
4. 综合评分 > 60
5. 护城河评分 > 70

**输出示例**：
```
推荐股票（价值型）：
1. 贵州茅台 (600519.SH) - 评分: 85
   - 理由: 护城河深厚，ROE 稳定，股息率 3.5%
   - 建议配置: 30%
```

---

### `growth` - 成长型###

**适合**：成长投资者、追求高收益

**筛选逻辑**：
1. 营收增速 > 20%
2. 利润增速 > 25%
3. PEG < 1.5
4. 综合评分 > 65

**输出示例**：
```
推荐股票（成长型）：
1. 宁德时代 (300750.SZ) - 评分: 88
   - 理由: 营收增速 35%，利润增速 42%，PEG 1.2
   - 建议配置: 25%
```

---

### `balanced` - 均衡型###

**适合**：大多数投资者、均衡配置

**筛选逻辑**：
1. 综合评分 TOP 3
2. 评分 > 65
3. 风险适中

**输出示例**：
```
推荐股票（均衡型）：
1. 贵州茅台 (600519.SH) - 评分: 85
2. 五粮液 (000858.SZ) - 评分: 78
3. 泸州老窖 (000568.SZ) - 评分: 75
   - 建议配置: 30% / 25% / 20% (合计 75%)
```

---

### `aggressive` - 激进型###

**适合**：激进投资者、短线交易

**筛选逻辑**：
1. 游资偏好（龙虎榜常客）
2. 技术突破（均线多头排列）
3. 综合评分 > 60

**输出示例**：
```
推荐股票（激进型）：
1. 某某妖股 (002XXX.SZ) - 评分: 72
   - 理由: 游资活跃，技术突破，短线机会
   - 建议配置: 10% (严控风险)
```

---

## 配置比例建议##

系统会根据评分自动计算建议配置比例：

```
总评分 = sum(股票评分)
股票 1 权重 = 评分 1 / 总评分
建议配置 = 权重 × 100%
```

**示例**：
- 股票 A 评分: 85
- 股票 B 评分: 78
- 股票 C 评分: 75
- 总评分: 238
- 配置比例:
  - A: 85/238 = 35.7% ≈ 35%
  - B: 78/238 = 32.8% ≈ 35%
  - C: 75/238 = 31.5% ≈ 30%

## 使用场景##

- **想投资某个行业，但不知道选哪只股票**
  → 用这个命令从行业中筛选推荐股票

- **想根据自己风格选择股票**
  → 选择不同的 `--criteria` 标准

- **想快速了解行业龙头**
  → 用默认的 `balanced` 标准，看综合评分 TOP 3

---

## 注意事项##

1. **行业名称要准确**
   - ✅ 正确："白酒"、"新能源汽车"、"半导体"
   - ❌ 错误："酒"、"汽车"、"芯片"

2. **筛选结果仅供参考**
   - 系统推荐基于量化评分
   - 最终决策需要结合自己的判断

3. **配置比例要灵活**
   - 系统建议是参考
   - 实际配置要根据自己的风险承受能力调整
