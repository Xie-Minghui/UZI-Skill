# 对比多个行业

对比多个行业，返回排名和最佳行业推荐。

## 用法

```
/stock-deep-analyzer:compare-industries <行业1>,<行业2>,<行业3>
```

## 参数

- `<行业列表>`: 行业名称列表（逗号分隔）

## 示例

```
/stock-deep-analyzer:compare-industries 白酒,新能源,医药
/stock-deep-analyzer:compare-industries 半导体,新能源汽车,生物医药
```

## 输出

- **行业排名**（按综合评分排序）
  1. 白酒 - 评分: 80 (值得关注)
  2. 新能源 - 评分: 65 (观望)
  3. 医药 - 评分: 50 (观望)

- **最佳行业**（评分最高的行业）
- **每个行业的简评**
  - 行业概览
  - 推荐股票数量
  - 主要风险

## 报告内容

生成的 HTML 报告包含：

1. **行业对比表格**
   - 行业名称
   - 综合评分
   - 评级
   - TOP 推荐股票

2. **行业详情**
   - 每个行业的详细分析
   - 评委对行业的看法
   - 风险提示

3. **投资建议**
   - 推荐最佳行业
   - 该行业的 TOP 3 股票
   - 配置建议

## 使用场景

- **想投资但不确定哪个行业**
  → 用这个命令对比 3-5 个候选行业，选择最佳行业

- **想分散投资到不同行业**
  → 看排名，选择 TOP 3 行业分别配置

- **想验证自己的行业判断**
  → 对比自己看好的行业和系统推荐的行业

## 注意事项

1. **行业数量建议 3-5 个**
   - ✅ 推荐：3-5 个行业（分析时间合理）
   - ⚠️ 注意：超过 5 个行业分析时间较长

2. **分析时间**
   - 对比 3 个行业：约 15-25 分钟
   - 对比 5 个行业：约 25-40 分钟

3. **行业选择建议**
   - 选择相关性低的行业（分散风险）
   - 例如：白酒 + 新能源 + 医药（不同赛道）

## 相关命令

- `/stock-deep-analyzer:analyze-industry` - 分析单个行业
- `/stock-deep-analyzer:screen-industry` - 从行业中筛选股票
- `/stock-deep-analyzer:recommend-portfolio` - 智能投顾（自动选择最佳行业）
