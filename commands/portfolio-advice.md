# 组合诊断#

针对自己的持仓组合，给出买入/卖出/持有建议。#

## 用法##

### 方式 1：从 CSV 文件读取###

```
/stock-deep-analyzer:portfolio-advice --file <CSV 文件路径>
```

**CSV 格式**：
```
ticker,shares,avg_cost
600519.SH,100,1800
000858.SZ,200,120
```

### 方式 2：交互式输入###

```
/stock-deep-analyzer:portfolio-advice
```

然后按提示输入持仓信息。

---

## 参数##

- `--file`: CSV 文件路径（可选，不提供则交互式输入）

---

## CSV 格式说明##

### 标准格式（有表头）###

```csv
ticker,shares,avg_cost
600519.SH,100,1800
000858.SZ,200,120
```

### 中文格式（有表头）###

```csv
股票代码,持股数量,平均成本
600519.SH,100,1800
000858.SZ,200,120
```

### 无表头格式###

```csv
600519.SH,100,1800
000858.SZ,200,120
```

系统会自动检测是否有表头。

---

## 输出##

- **组合健康度评分**（0-100 分）
- **每只股票的操作建议**：
  - `buy`: 建议买入/加仓
  - `sell`: 建议卖出/减仓
  - `hold`: 建议持有/观望
- **目标仓位**（系统建议的配置比例）
- **盈亏分析**（当前价 vs 成本价）
- **风险提示**

---

## 操作建议逻辑##

系统根据**综合评分**和**盈亏状态**生成建议：

| 综合评分 | 盈亏状态 | 建议 |
|---------|---------|-------|
| ≥ 80 | 盈利 > 20% | `hold` (止盈观望) |
| ≥ 80 | 盈利 ≤ 20% | `buy` (加仓) |
| < 50 | 盈利 | `sell` (止盈) |
| < 50 | 亏损 | `sell` (止损) |
| 50-80 | - | `hold` (观望) |

---

## 报告内容##

生成的 HTML 报告包含：

1. **组合概览**
   - 组合健康度评分
   - 持仓数量
   - 建议操作统计（买入 X / 卖出 Y / 持有 Z）

2. **持仓分析表格**
   - 股票名称（代码）
   - 持股数量
   - 平均成本
   - 当前价格
   - 盈亏百分比
   - 综合评分
   - **操作建议**（买入/卖出/持有）
   - **目标仓位**
   - **理由**

3. **风险提示**
   - 持仓集中度风险
   - 行业分布风险
   - 市场风险

---

## 使用场景##

- **想了解自己的持仓是否合理**
  → 用这个命令诊断组合，获取操作建议

- **想优化自己的仓位配置**
  → 看系统建议的目标仓位，调整持仓

- **想止损或止盈**
  → 看系统的操作建议，决定卖出时机

---

## 注意事项##

1. **CSV 文件要准确**
   - ✅ 正确：`600519.SH,100,1800`
   - ❌ 错误：`茅台,100,1800`（要用股票代码）

2. **建议仅供参考**
   - 系统建议基于量化评分
   - 最终决策需要结合自己的判断

3. **及时更新 CSV**
   - 如果持仓有变化，要更新 CSV 文件
   - 或者重新交互式输入
