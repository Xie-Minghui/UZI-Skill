# 智能投顾（选择行业 + 筛选股票 + 配置资金）#

完整投资决策流程：**先选择行业 → 再选择股票 → 最后配置资金**。#

## 用法##

```
/stock-deep-analyzer:recommend-portfolio <行业1>,<行业2> [--budget <预算>] [--risk <风险偏好>]
```

---

## 参数##

- `<行业列表>`: 候选行业列表（逗号分隔）
- `--budget`: 预算（元），默认 100000（10 万元）
- `--risk`: 风险偏好（可选）
  - `conservative`: 保守
  - `moderate`: 稳健（**默认**）
  - `aggressive`: 激进

---

## 示例##

```
/stock-deep-analyzer:recommend-portfolio 白酒,新能源,医药
/stock-deep-analyzer:recommend-portfolio 白酒,新能源,医药 --budget 200000
/stock-deep-analyzer:recommend-portfolio 白酒,新能源 --risk aggressive
```

---

## 执行流程##

### 步骤 1：行业筛选（功能 1）###

系统会：
1. 分析每个候选行业
2. 计算行业综合评分
3. 返回行业排名
4. 选择**最佳行业**

**输出示例**：
```
📊 步骤 1/4: 行业筛选...
✅ 最佳行业: 白酒 (评分: 80)
```

---

### 步骤 2：个股挖掘（功能 2）###

系统会：
1. 从最佳行业中选择 TOP 10 龙头股
2. 根据风险偏好选择筛选标准
3. 筛选推荐股票（TOP 3）
4. 生成配置建议

**筛选标准映射**：
| 风险偏好 | 筛选标准 | 说明 |
|---------|---------|-------|
| `conservative` | `value` | 价值型（低 PE + 高股息） |
| `moderate` | `balanced` | 均衡型（综合评分 TOP 3） |
| `aggressive` | `growth` | 成长型（高增速） |

**输出示例**：
```
📊 步骤 2/4: 从 '白酒' 中筛选个股...
✅ 推荐股票: 3 只
   1. 贵州茅台 (600519.SH) - 评分: 85
   2. 五粮液 (000858.SZ) - 评分: 78
   3. 泸州老窖 (000568.SZ) - 评分: 75
```

---

### 步骤 3：生成组合配置###

系统会：
1. 根据评分分配权重
2. 留出**现金储备**（根据风险偏好）
3. 计算每只股票的投资金额
4. 生成资金配置方案

**现金储备比例**：
| 风险偏好 | 现金储备 | 股票投资 |
|---------|---------|---------|
| `conservative` | 30% | 70% |
| `moderate` | 20% | 80% |
| `aggressive` | 10% | 90% |

**输出示例**：
```
📊 步骤 3/4: 生成组合配置...
✅ 资金配置方案:
   总投资: ¥100,000
   股票投资: ¥80,000
   现金储备: ¥20,000
   - 贵州茅台: ¥30,000 (30%)
   - 五粮液: ¥25,000 (25%)
   - 泸州老窖: ¥25,000 (25%)
```

---

### 步骤 4：生成完整报告###

系统会生成一份完整的 HTML 报告，包含：
1. 行业选择逻辑
2. 个股推荐理由
3. 资金配置方案
4. 预期收益和风险
5. 操作建议

**输出示例**：
```
📊 步骤 4/4: 生成投资建议报告...
✅ 投资建议生成完成
   报告路径: reports/recommend_20260705_213000/index.html
```

## 使用场景##

- **想开始投资，但不知道从哪里入手**
  → 用这个命令获取完整的投资建议（行业 + 股票 + 配置）

- **有一笔资金，想分散投资**
  → 设置合理的预算，让系统配置组合

- **想根据自己的风险偏好定制投资方案**
  → 选择合适的 `--risk` 参数

---

## 注意事项##

1. **行业数量建议 3-5 个**
   - ✅ 推荐：3-5 个行业（分析时间合理）
   - ⚠️ 注意：超过 5 个行业分析时间较长

2. **预算要合理**
   - 系统会根据预算和评分分配资金
   - 确保预算符合自己的实际情况

3. **风险偏好要选择合适**
   - `conservative`: 适合临近退休、稳健型投资者
   - `moderate`: 适合大多数投资者（默认）
   - `aggressive`: 适合年轻投资者、追求高收益

4. **建议仅供参考**
   - 系统建议基于量化评分
   - 最终决策需要结合自己的判断
