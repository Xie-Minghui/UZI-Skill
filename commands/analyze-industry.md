# 分析单个行业

分析指定行业，给出行业评级和投资建议。

## 用法

```
/stock-deep-analyzer:analyze-industry <行业名称>
```

## 参数

- `<行业名称>`: 行业名称（如"白酒"、"新能源"、"医药"）

## 示例

```
/stock-deep-analyzer:analyze-industry 白酒
/stock-deep-analyzer:analyze-industry 新能源
/stock-deep-analyzer:analyze-industry 医药生物
```

## 输出

- **行业综合评分**（0-100 分）
- **行业评级**（值得重仓 / 可以蹲一蹲 / 观望 / 谨慎 / 回避）
- **TOP 3 推荐股票**（股票名称、代码、评分、推荐理由）
- **看多理由**（为什么看好这个行业）
- **看空理由**（风险提示）
- **风险提示**（行业特有风险）

## 报告内容

生成的 HTML 报告包含：

1. **行业概览**
   - 行业指数表现（1 日、1 周、1 月、年初至今）
   - 估值分位（PE/PB 历史分位）
   - 行业增速（营收增速、利润增速）

2. **成分股分析**
   - TOP 10 龙头股对比表格
   - 每只股票的评分和推荐理由
   - 护城河分析

3. **评委视角**
   - 66 位评委对行业的整体看法
   - 多方观点（为什么看好）
   - 空方观点（为什么看空）

4. **投资建议**
   - 推荐买入的股票列表
   - 建议配置比例
   - 预期收益和风险

## 使用场景

- **想投资某个行业，但不知道选哪只股票**
  → 用这个命令分析行业，获取 TOP 3 推荐股票

- **想了解某个行业是否值得投资**
  → 看行业评级和评分，快速判断

- **想对比多个行业，选择最佳投资方向**
  → 配合 `/stock-deep-analyzer:compare-industries` 使用

## 注意事项

1. **行业名称要准确**
   - ✅ 正确："白酒"、"新能源汽车"、"半导体"
   - ❌ 错误："酒"、"汽车"、"芯片"

2. **分析时间**
   - 快速模式（`--depth lite`）：1-2 分钟
   - 标准模式（默认）：5-8 分钟
   - 深度模式（`--depth deep`）：15-20 分钟

3. **报告保存位置**
   - 默认：`reports/industry_<行业名称>_<时间戳>/index.html`
   - 可用 `--output-dir` 指定输出目录

## 相关命令

- `/stock-deep-analyzer:compare-industries` - 对比多个行业
- `/stock-deep-analyzer:screen-industry` - 从行业中筛选股票
- `/stock-deep-analyzer:recommend-portfolio` - 智能投顾（选择行业 + 筛选股票）
